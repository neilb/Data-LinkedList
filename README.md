Data::LinkedList
===========

A Perl implementation of GNU classpath LinkedList class (doubly linked list).

This module provide an object orieted interface to a doubley linked list data structure.

Installation:

    perl Makefile.PL
    make test install clean all

Prerequisites:

    Test::Simple    => 0.98
    Test::Exception => 0.31
    Storable        => 2.34
    Iterator::Util  => 0.02
    Carp            => 1.26
